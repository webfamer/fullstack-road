# 141. 环形链表

> 来源：https://www.yuque.com/xiumubai/doc/txmhmm83g9gign92

1. 题目描述
给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

示例 1：
​912输入：head = [3,2,0,-4], pos = 1输出：true解释：链表中有一个环，其尾部连接到第二个节点。

2. 解题思路
我们只需要对每个遍历过的节点进行标记，后面如果再遇到它，说明有环，就直接返回true
