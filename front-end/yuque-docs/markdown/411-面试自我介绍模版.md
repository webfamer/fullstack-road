# 面试自我介绍模版

> 来源：https://www.yuque.com/xiumubai/doc/htiitmf88qnf7ege

一般自我介绍只需要包含 3 部分内容：
●基本信息：你叫xxx，xx毕业，专业，几年前端工作经验
●职业经历：我在xxx、xxx公司(最好是一线互联网大厂)呆过，现在在xx公司担任高阶(前端)；
●特长(最好是技术特长)：
○对xx方面特别感兴趣 or 擅长，曾经做过xxx技术项目(轮子、性能优化、解决过复杂问题)；
○曾经做过什么技术优化(性能、工程化、质量 & 稳定性、安全等)
○主导 or 参与过什么知名的开源项目
○当下在做什么，特别擅长的点
○比较成体系的团队管理经验；
核心：简单明了，把基本情况和核心竞争力传达清楚

案例一：
	我叫 xxx，15 年毕业与 xxx 学校，毕业后开始做前端，之前在 腾讯、阿里 待过，现在在xxx公司xx业务线做前端 leader；我个人一是对工程化方向非常感兴趣，曾经基于 webpack 做过一个小程序预编译工具；二是对 IDE 技术感兴趣，目前正在主导 xxx IDE 方向的工作。

案例二：
	我叫 xxx，15 年毕业开始做前端，之前在 腾讯、阿里 待过，现在在xxx公司xx业务线做前端leader；我个人有多年管理经验，曾经 0-1 搭建过一个 xx 人团队(担任xx人团队管理者)，在团队内持续推进规范、技术分享、梯队、激励、晋升各方面建设

案例三：
我叫 xxx，15年毕业开始做前端，之前在 腾讯、阿里 待过，现在在xxx公司xx业务线担任高级前端工程师，我个人对xx非常感兴趣，曾经基于 xx 做过xxx(性能优化、轮子、blabla)，现在主要负责xx业务。
