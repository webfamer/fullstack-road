# 92. 反转链表 II

> 来源：https://www.yuque.com/xiumubai/doc/sofuu01gqhgr19ow

1. 题目描述
反转从位置 m 到 n 的链表。请使用一趟扫描完成反转。

说明：1 ≤ m ≤ n ≤ 链表长度。

示例:
​912输入: 1->2->3->4->5->NULL, m = 2, n = 4输出: 1->4->3->2->5->NULL2. 解题思路
这个题最主要的就是将第m到n的节点进行饭反转，然后将m-1节点指向n，让n+1节点指向m，这样就实现了链表的局部反转。
