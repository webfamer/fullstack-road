# 242. 有效的字母异位词

> 来源：https://www.yuque.com/xiumubai/doc/zbhopuay1kcm3mml

1. 题目描述
给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

输入: s = "anagram", t = "nagaram"输出: true示例 2:
输入: s = "rat", t = "car"输出: false说明:
你可以假设字符串只包含小写字母。

进阶:
如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？
2. 解题思路
（1）首先将字符串转换为数组
（2）将数组元素进行排序
（3）将排序好的数组元素转化为字符串，并比较是否相等
3. 代码实现
​JavaScriptRun CodeCopy991234567891011121314/** * @param {string} s * @param {string} t * @return {boolean} */var isAnagram = function(s, t) {   var sArr = s.split('');   var tArr = t.split('');
   sArr.sort();   tArr.sort();
   return sArr.join('')===tArr.join(''); };4. 提交结果
