# 739. 每日温度

> 来源：https://www.yuque.com/xiumubai/doc/ego6hwaafr5vpviv

1. 题目描述
请根据每日 气温 列表，重新生成一个列表。对应位置的输出为：要想观测到更高的气温，至少需要等待的天数。如果气温在这之后都不会升高，请在该位置用 0 来代替。

例如，给定一个列表 temperatures = [73, 74, 75, 71, 69, 72, 76, 73]，你的输出应该是 [1, 1, 4, 2, 1, 1, 0, 0]。

提示：气温 列表长度的范围是 [1, 30000]。每个气温的值的均为华氏度，都是在 [30, 100] 范围内的整数。
2. 解题思路
对于这个题目，能直接想到的就是暴力求解，设置两层循环，求出相应的值。我们需要设置一个flag来排除之后温度不会再升高的情况。

当然，我们也可以看到，这种方法需要两次循环，时间复杂度较高，所以，考虑使用栈来解决这个问题：

我们需要设置一个定长的结果数组，然后把数组的值都设置为0。
然后对T进行遍历，如果比栈顶元素的值大，就将栈顶元素出栈，并求出两者的差，放入结果数组中。反之则将元素i放入栈中。
