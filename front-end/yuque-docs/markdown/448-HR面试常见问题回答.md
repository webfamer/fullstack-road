# HR面试常见问题回答

> 来源：https://www.yuque.com/xiumubai/doc/sdfe0hagr8bua353

问题1:请自我介绍一下
回答重点:能在30秒内展现和岗位的匹配度
必过公式:基本信息+工作经历+胜任力+性格特征
参考回答:不要读简历，基本信息简单说，工作经历只讲一个最成功的项目经历，胜任力要围绕着招聘JD讲自己最突出的技能，性格特征要突出与众不同但又非常积极的一面。
问题2:请说一个你收获最大的项目
回答重点:逻辑清晰，重点突出回答公式:STAR法则
参考回答:说一个你最有成就感的项目，项目不一定要大，但一定要有意义，要说清楚项目的来龙去脉和自己在其中发挥的核心作用。用SATR法则说明你的项目背景、你的目标、你的动作和项目结果。
问题3:你对我们公司了解多少
回答重点:提前做功课
回答公式:公司战略+个人看法
参考回答:我了解到咱们公司的企业战略和价值观是什么什么，这也恰好和我个人的职业目标和价值观相吻合我认为贵公司的战略真正是从商业的底层逻辑出发，致力于为客户提供最有效的解决方案，而我自己就是一个解决问题导向的人，相信我也会很快融入到公司的企业文化中。
问题4:你认为自己最大的缺点是什么
回答重点:不要讲跟目标岗位相关的缺点回答公式:缺点+原因+改正情况
参考回答:前领导总说我工作太着急，后来我发现是自己的时间管理有问题，为此我还专门报了一门时间管理的课，结合业务实操后，发现自己在这方面提升明显，相信不远的将来就可以得到彻底的改善。
问题5:五年内你对自己有什么职业规划吗
回答重点:说软规划，不说硬指标
回答公式:对业务的思考+自己的业务价值观+目标
参考回答:不要说职位、薪资等硬指标，要体现出对业务的专一。比如我希望在五年内始终如一的服务某一类客户，深刻了解行业和客户需求，能站在更高的角度上为客户解决问题，最终成为行业专家。
问题6:你接受加班吗回答重点:能接受
回答公式:表明态度+结果导向
参考回答:愿意接受加班，如果是自己职责范围的工作那就不算加班，但我一般都会结果导向的看问题，加班只是过程，解决问题，降本增效才是工作的目的，在工作中我会多去寻找高效的工作方法，争取讲无效工作压缩到最少。
问题7:来了新公司，你如何快速进入适应工作并进入状态。
回答重点:考虑问题全面，重视调研回答公式:了解业务+明确目标
参考回答:首先我会与领导和高频对接人充分沟通业务目标，明确自己的职责和角色;其次我会充分利用公司的知识资源，让自己快速积累业务认知。最后我会通过拆解整体业务，按照工作优先级四象限找出最重要紧急的工作优先处理，同时在工作中积累业务感觉。
问题8: 你能给我们公司带来什么?
回答重点:复盘优势，数据验证
回答公式:重申优势+前岗位的业绩
参考回答:我的核心优势是什么，这个优势，让我在前公司的某某项目中，获得了什么样的成绩，这个成绩比过往高出多少，比同事高出多少，比同行高出多少。希望我这个优势也能应用到咱们公司来。
问题9:你对薪资的要求
回答重点:求上得中
回答公式:薪资目标+原因
参考回答:我的薪资目标是多少多少，主要是因为，此前我做过调研，同行业下跟我同级别的人的薪资大概是多少多少，我认为自己的能力和价值应该在行业内能排到30%，所以我希望在这个基础上有20%的溢价。
问题10: 你领导同事对你的评价如何
回答重点:围绕个人品性回答回答公式:个人品性+举例
参考回答:我领导和同事都觉得我是一个靠谱的人，说到就能做到，而且经常能超预期完成任务。比如领导让我做某个业务的客户梳理，我不仅能按照领导要求完成工作，还能提出自己的见解和解决方案。
问题11:你现在有其他的offer吗
回答重点:表明对该公司更有兴趣
回答公式: 有offer+对现公司有兴趣+原因
参考回答:确实有公司给我offer，但是我更想来咱们公司，因为咱们公司在业界有口皆碑，这个岗位也是我想做的，而且通过跟您沟通，您给我的感受也非常好，这也能侧面反映咱们公司的人文关怀
问题12: 如果你被录用后，工作了一段时间发现自己不适合这个岗位，你会怎么办
回答重点:有解决问题的态度
回答公式:分析原因+给出方案+表达态度
参考回答: 短期的不适应可能是业务不熟导致的，这也是正常现象，我会尽快熟悉业务，融入环境，深入了解业务本质，相信只需要两个月的时间，我就能尽快调整状态。
问题13:你有什么问题要问我吗
回答重点:要表现出对岗位的兴趣，体现上进心，有想法有欲望
回答公式:明确兴趣+了解岗位职责+了解晋升机制
参考回答:我仔细看过了招聘JD，我对这个岗位非常有兴趣，且我认为自己能够胜任这个岗位，不过招聘JD上饿某某职责，具体的工作内容我还是不是很清晰，您能帮我再解释一下吗?
问题14:如果我们没有录取你，你下步有什么打算
回答重点:展示锲而不舍的精神
回答公式:认可结果+找出不足+努力方向
参考回答:如果没被贵公司录用，说明我个人的能力与公司岗位的匹配度还有一定偏差，未来我会分析原因，总结问题，然后针对性的对自己提升，当自己满足了公司的招聘条件后，还会投递公司的岗位。
问题15: 你的兴趣爱好是什么
回答重点:体现上进心
回答公式:兴趣爱好+提炼能力+举例
参考回答:我的兴趣爱好是看书，看书不仅可以增长见识，还可以让自己拥有解决问题的方法，我每年的阅读量大约是50本书，我最近读的一本书是《XXX》，我从这本书中学到了XXX
问题16: 你为什么从上家公司离职
回答重点:避免一切负面情绪
回答公式:原因+自己的考虑+目标
参考回答:因为上一家公司进行了组织结构的调整，导致自己的核心能力有点发挥不出来，考虑到自己的职业规划，我决定换工作，希望能够找一个像贵公司这样的大平台和更符合自己职业规划的岗位，恰好这个岗位就是我想做的。
问题17:当你和领导意见不一致，你会怎么办
回答重点:目标导向
回答公示:分析原因+明确目标+求同存异
参考回答:如果我和领导意见不一致，很有可能是我们对业务目标的理解不同，这时候需要深入的沟通，先在目标层面达成一致，然后分析各自的实现路径，主动向领导说出自己的顾虑，但最终还是会尊重领导的意见。
