# 701. 二叉搜索树中的插入操作

> 来源：https://www.yuque.com/xiumubai/doc/guaq4t91ffsoqf9r

1. 题目描述
给定二叉搜索树（BST）的根节点和要插入树中的值，将值插入二叉搜索树。 返回插入后二叉搜索树的根节点。 保证原始二叉搜索树中不存在新值。

注意，可能存在多种有效的插入方式，只要树在插入后仍保持为二叉搜索树即可。 你可以返回任意有效的结果。

例如,

给定二叉搜索树:
​912345    4   / \  2   7 / \1   3和 插入的值: 5
你可以返回这个二叉搜索树:
​912345     4   /   \  2     7 / \   /1   3 5或者这个树也是有效的:
​91234567     5   /   \  2     7 / \   1   3     \      42. 解题思路
使用递归的迭代的方法实现。
