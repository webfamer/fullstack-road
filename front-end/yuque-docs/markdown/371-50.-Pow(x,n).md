# 50. Pow(x,n)

> 来源：https://www.yuque.com/xiumubai/doc/err6i70bgwts09zq

1. 题目描述
实现 pow(x, n) ，即计算 x 的 n 次幂函数（即，xn）。

示例 1：
​912输入：x = 2.00000, n = 10输出：1024.00000示例 2：
​912输入：x = 2.10000, n = 3输出：9.26100示例 3：
​9123输入：x = 2.00000, n = -2输出：0.25000解释：2-2 = 1/22 = 1/4 = 0.25提示：
●-100.0 < x < 100.0
●-231 <= n <= 231-1
●-104 <= xn <= 104
2. 解题思路

这道题直接的思路就是暴力计算，判断n的正负，如果为正就让底数x直接累乘，如果为负就让底数的倒数1/x累乘。

但是超出了时间限制，可以使用二分法进行优化：

我们可以进行折半计算，每次将n缩小一半，通过递归最终获得结果。注意，当n为奇数时，需要多乘一次x的值。

时间复杂度：O(log2n)，空间复杂度：O(log2n)
3. 代码实现
暴力：
