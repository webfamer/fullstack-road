# 700. 二叉搜索树中的搜索

> 来源：https://www.yuque.com/xiumubai/doc/lqly0hpv5suldgw1

1. 题目描述
给定二叉搜索树（BST）的根节点和一个值。 你需要在BST中找到节点值等于给定值的节点。 返回以该节点为根的子树。 如果节点不存在，则返回 NULL。

例如，

给定二叉搜索树:
​912345    4   / \  2   7 / \1   3和值: 2
你应该返回如下子树:
​9123  2      / \   1   3在上述示例中，如果要找的值是 5，但因为没有节点值为 5，我们应该返回 NULL。
2. 解题思路
可以用迭代和递归的方法实现
