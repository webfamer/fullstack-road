# 69. x的平方根

> 来源：https://www.yuque.com/xiumubai/doc/oh03fq7bek5glby1

1. 题目描述
实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

这里可以使用二分法来解决这个问题。我们知道，所有数字的平方根肯定比它一半是小的，所以我们可以直接将right设置为x / 2 这里我们使用使用>>1来求一个数的一半或者两个数的平均值，他会比Math的方法快很多。不断缩小left和right之间的范围，直至找到平方根。
3. 代码实现
​JavaScriptRun CodeCopy9912345678910111213141516171819/** * @param {number} x * @return {number} */var mySqrt = function(x) {    if(x < 2) return x
    let left = 1 , right = x >>1, mid
    while(left <= right){        mid = (left + right) >>1
        if(mid * mid === x){            return mid        }        mid * mid < x ? left = mid + 1 : right = mid - 1    }    return right};4. 提交结果
