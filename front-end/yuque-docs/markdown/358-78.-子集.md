# 78. 子集

> 来源：https://www.yuque.com/xiumubai/doc/to8nv13xvmad5ac5

1. 题目描述
给定一组不含重复元素的整数数组 nums，返回该数组所有可能的子集（幂集）。

说明：解集不能包含重复的子集。

示例:

输入: nums = [1,2,3]
输出:
​9912345678910[  [3],  [1],  [2],  [1,2,3],  [1,3],  [2,3],  [1,2],  []]2. 解题思路
这里求得是数组的子集，也就是说，在所有的子集中，每个元素㕛两种状态：出现和不出现。所以我们只要保证将将每个元素遍历，让他出现或者不出现。

看下面的图示，取[1,2,3]的子集：
![](https://cdn.nlark.com/yuque/0/2021/png/1500604/1621585855186-f09edc6c-723c-4565-8b30-8280ae0763a7.png)
只需要能不断往下遍历，然后回溯就可以了。这里使用index来指元素的位置。
3. 代码实现
​JavaScriptRun CodeCopy9912345678910111213141516171819202122/** * @param {number[]} nums * @return {number[][]} */var subsets = function(nums) {    const res = []
    const dfs = (index, list) =>{        if(index === nums.length){            res.push([...list])            return        }        // 选择这个元素，继续遍历，然后进行回溯        list.push(nums[index])        dfs(index + 1, list)        list.pop()        // 不选择这个元素，直接往下遍历        dfs(index + 1, list)    }    dfs(0, [])    return res};4. 提交结果
