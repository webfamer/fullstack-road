# 216. 组合总和 III

> 来源：https://www.yuque.com/xiumubai/doc/zmkfzq37c7dkm3if

1. 题目描述
找出所有相加之和为 n 的 k 个数的组合。组合中只允许含有 1 - 9 的正整数，并且每种组合中不存在重复的数字。

说明：
●所有数字都是正整数。
●解集不能包含重复的组合。
