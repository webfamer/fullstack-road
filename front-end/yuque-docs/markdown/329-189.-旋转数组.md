# 189. 旋转数组

> 来源：https://www.yuque.com/xiumubai/doc/iiv13xuwvlibmyuy

1. 题目描述
给定一个数组，将数组中的元素向右移动 k 个位置，其中 k 是非负数。

进阶：
●尽可能想出更多的解决方案，至少有三种不同的方法可以解决这个问题。
●你可以使用空间复杂度为 O(1) 的 原地 算法解决这个问题吗？

●1 <= nums.length <= 2 * 104

2. 解题思路
对于这道题目，其实很简单，如果k大于数组的长度，那么k的倍数的部分相当于原地不动，不需要操作，所以需要对数组的长度进行取余操作，获取实际操作的步数。

所以只需要遍历数组，从数组最后取出元素放在数组的开头即可。

复杂度分析：
●时间复杂度：O(n)，其中n是遍历数组的次数
●空间复杂度：O(1)，我们不需要额外的空间去储存变量，只需在数组原基础上进行操作。
