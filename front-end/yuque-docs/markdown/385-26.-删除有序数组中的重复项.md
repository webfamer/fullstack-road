# 26. 删除有序数组中的重复项

> 来源：https://www.yuque.com/xiumubai/doc/alhvtb5z4etpvgy0

1. 题目描述
给你一个有序数组 nums ，请你[原地](http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95) 删除重复出现的元素，使每个元素 只出现一次 ，返回删除后数组的新长度。
不要使用额外的数组空间，你必须在 [原地](https://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)修改输入数组 并在使用 O(1) 额外空间的条件下完成。

说明:

请注意，输入数组是以「引用」方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

2. 解题思路
对于这道题目，给出的数组是有序的，所以我们就可以直接遍历数组，比较相邻的元素，如果两者相同，删除一个即可。需要注意的是，删除一个之后，需要将当前索引值减一，不然这个元素会被跳过去。

复杂度分析：
●时间复杂度：O(n)，其中n是数组的长度，我们需要遍历整个数组来进行数组元素的对比；
●空间复杂度：O(1)，这里我们需要的额外空间为常量。
3. 代码实现
