1参与 Taro 框架开发，同时在跨端和全栈等技术方向攻克难题；

职位要求
1本科及以上学历，优先考虑计算机或软件工程专业，必须具备扎实的技术基础；
2具备对技术充满好奇心，乐于学习和分享的态度，积极乐观，认真负责，并具备善于钻研的能力；
3具备良好的服务意识和责任心，具备团队沟通和协作能力，能够发现并解决问题；
4熟练掌握各种 Web 前端技术，至少熟悉一种 Vue / React 常见 MV* 前端框架，并对其关键细节有较深的了解；
5熟悉掌握至少一种 RN / Flutter / Weex / Hybrid H5 / Taro 跨端框架，并具有相关项目开发经验；
6对前端工程化相关技术非常熟悉，熟练掌握 Webpack / Vite 等前端构建工具的使用。
