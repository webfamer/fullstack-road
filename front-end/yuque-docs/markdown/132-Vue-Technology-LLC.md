职位
全职开源开发者
职责内容
参与 Vue.js 官方项目的开发和维护。包括但不限于：
●Vue 相关仓库的日常 bug 修复和新功能研发（core, router, vuex, CLI, devtools...)
●基础设施／工作流改进，比如：
○搭建自动发布系统（按周期发布 canary / nightly release)
○搭建 Vue 发布周期可视化应用
○搭建 Vue RFC（新功能提案） 可视化／投票／管理应用
○搭建跨仓库回归测试系统
○开发更完善的 Vue 性能 benchmarks 和通过 CI 收集数据的可视化系统
总的来说，虽然 Vue 是一个前端框架，但工作内容不仅限于 “前端”，你会有机会接触到从框架底层到应用开发、构建工具、工程化、开源协作各方面的内容。
能力要求
●对 JavaScript 语言有深入了解，熟悉 Node.js，熟悉 Unix 系统 (macOS / Linux)。
●充满好奇心，对 JavaScript 生态有广泛的涉猎，对不同框架、工具、库都有自己的理解，不局限于自己的舒适区。
●英语能力过关，能够轻松阅读英文文档，流畅地用英文进行书面技术交流，能够直接看无字幕的外国技术演讲视频。口语能力不作要求，但口语能力优秀加分。
●钻研能力强，知道怎么看文档，怎么用 Google，敢于看源码。
●对资历没有硬性要求，应届生也可以，但必须符合上述条件。
●加分：熟悉 TypeScript / Flow
●加分：有自己的或是作为核心开发者参与的非玩具开源项目。
●加分：给 Vue 或是其他知名开源项目贡献过 non-trivial PR。
