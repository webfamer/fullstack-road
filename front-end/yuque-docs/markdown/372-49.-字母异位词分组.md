# 49. 字母异位词分组

> 来源：https://www.yuque.com/xiumubai/doc/zi9023uv2gqy2dzg

1. 题目描述
给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排列不同的字符串。

示例:
​JavaScriptRun CodeCopy91234567输入: ["eat", "tea", "tan", "ate", "nat", "bat"]输出:[  ["ate","eat","tea"],  ["nat","tan"],  ["bat"]]说明：
●所有输入均为小写字母。
●不考虑答案输出的顺序。
2. 解题思路
对于这道题，最常规的思路就是，先遍历数组，将每个元素字母都进行排序，将排序后的字符串作为key值，将key值一样的数据存在用以数组中，最后按照题目的要求返回数组。
3. 代码实现
​JavaScriptRun CodeCopy991234567891011121314151617181920/** * @param {string[]} strs * @return {string[][]} */var groupAnagrams = function(strs) {    const obj = {} // 用来将字母异位数进行分类存储    const arr = [] // 用来存放最后的结果    for(let i = 0; i< strs.length; i++){        // 将每个字母异位数进行拷贝，并将内部的字母进行排序，组合成一个字符串，并作为obj中的key值        const unit = Array.from(strs[i]).sort().join('')        if(!obj[unit]){            obj[unit] = []  // 创建该key值得数组        }        obj[unit].push(strs[i]) // 将与这个key值相符的放入对应的数组    }    for(const i in obj){        arr.push(obj[i]) // 将obj中的结果保存在arr中    }    return arr};4. 提交结果
