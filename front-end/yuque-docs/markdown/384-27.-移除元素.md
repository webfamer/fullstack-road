# 27. 移除元素

> 来源：https://www.yuque.com/xiumubai/doc/pa4wgv8vb1y6mobs

1. 题目描述
给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

说明:
为什么返回数值是整数，但输出的答案是数组呢?

请注意，输入数组是以「引用」方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

2. 解题思路
对于这道题目，只需要遍历数组，将与目标值相等的元素移除即可，当数组元素不等于目标元素时，就将res加一，最后res就是所求的结果值。

复杂度分析：
●时间复杂度：O(n)，其中n是数组的长度，需要遍历整个数组的元素；
●空间复杂度：O(1)，这里只需要初始化一个结果变量，它是一个常数；
3. 代码实现
​JavaScriptRun CodeCopy99123456789101112131415/** * @param {number[]} nums * @param {number} val * @return {number} */var removeElement = function(nums, val) {    let res = 0    for(let item of nums){        if(item !== val){            nums[res] = item            res ++        }    }    return res};双指针：
