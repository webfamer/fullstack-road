# 100. 相同的树

> 来源：https://www.yuque.com/xiumubai/doc/dux6p78clfep4hcb

1. 题目描述
给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

示例 1:
​91234567输入:       1         1          / \       / \         2   3     2   3
        [1,2,3],   [1,2,3]
输出: true示例 2:

        [1,2,1],   [1,1,2]
输出: false2. 解题思路
这个实际上也没啥好说的就是进行递归遍历两个树对应的节点，看看是否一致，一致的话就直接返回false。
