# Vue 面试知识点 Demo

这是配合 `../vue2-vue3面试速成.md` 的可运行 demo 项目。

## 覆盖内容

- Vue2 / Vue3 响应式差异
- `ref` / `reactive` / `toRefs`
- `computed` / `watch` / `watchEffect`
- `v-if` / `v-show`
- Vue3 多 `v-model`
- 生命周期和 `KeepAlive`
- Vue Router 权限菜单
- Pinia 与 `storeToRefs`
- WebSocket 高频数据和大屏性能优化思路
- Vibe Coding / AI 辅助开发工作流

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端里显示的地址，默认是：

```text
http://127.0.0.1:5173/
```

## 面试使用方式

每个 demo 都按三段记：

1. 面试官会怎么问。
2. 页面上怎么操作验证。
3. 你用一句话讲出结论。
