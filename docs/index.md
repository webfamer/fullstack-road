## 这不是普通文档转换

这套教程不是把几份 Markdown 手册简单转成网页。它的目标是把你正在补的能力组织成一条学习路径：

```txt
业务需求
  ↓
表结构设计
  ↓
SQL 与事务
  ↓
FastAPI 接口
  ↓
项目分层
  ↓
并发一致性
  ↓
练习与面试表达
```

你最终要获得的不是“看过 FastAPI 文档”，而是能独立完成一个后端业务模块，并能讲清楚自己的设计。
